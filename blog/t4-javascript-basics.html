Intro to Javascript

25th July, 2016

  • How does JavaScript compare to HTML and CSS?

  • HTML is uses tags to markup a web page, or give it structure. CSS gives the page style and implement the design and positional aspects, with some basic interactive elements. whereas Javascript is scripting language that in conjunction to HTML and CSS, adds interactive functions and elements.

  • Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.

  • Loop/ Control flow: Eating a meal; I fork food into mouth. Chew. Swollow. Repeat this till meal is finished. If I choke. Then die. The end. Control flow takes in account conditions and parameters, and when it is triggered, it runs through till a condition/outcome is met.

  • Explain the difference between accessing data from arrays and object literals.

  • Arrays are variables which can contain many values(numbers, strings or objects) at one time. To access data from an array you use ‘arrayName[ x ]’, arrays are 0 indexed, meaning the first value is set at 0 or [ 0 ] Objects contain properties, properties consist of a name/key and a value. There are two methods of accessing an object dot.notation objectName.key or objectName[“key”].
