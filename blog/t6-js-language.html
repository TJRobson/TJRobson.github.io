Javascript Language

05th August, 2016

  • Here are the uses for a few of the characters adopted in the Javascript syntax.

  • Parentheses ( ) - these are used to define a condition before a code block if( 1 + 2 = 3) { return “true”; }; or to call a function function addition(a, b) { return a + b; } and to call the function addition(5, 5); //25

  • Brackets [ ] - to declare an array array = [1, 2, 3 ] and to index into an array array[0] // 1

  • Braces { } - declare an object’s value pairs var shoe = {“eyelets”: 6, “upper”: “leather”, “size”: 8}; or open up a code block after a condition if(true) { console.log “It is true”};

  • Single ‘ ‘ & Double“ ” quotes - there almost the same thing. The are used to contain a string ‘I am a string’ or this is an empty string “ “ . “They are both used ‘when’ there is a need!”.
