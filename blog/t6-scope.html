Intro to Javascript

21st August, 2016

  • What 'scope' is, and how it works in JavaScript

  • Scope is the variables you have access to. In Javascript there are variables that are Global, these are declared outside of a function. All scripts and function can access it, making it global in scope.

  • Variables with Local scope are declared within a function. These variables are only accessible to the function in which they are defined is, as such they are local in scope.

  • In javascript it looks for variables first at a local level then works out to a global level.
