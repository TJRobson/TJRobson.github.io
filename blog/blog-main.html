Technical Blog Part 1 - Udacity Reflection

11th July, 2016


Can you think of an analogy to describe HTML, CSS, and the DOM? How would you describe it to your non-technical friend?

An analogy could be that of conversation. In conversation, words are strung together. Words are like containers, they have their own meanings and concepts. This is like HTML, with tags building structure. But words don’t live in isolation, their meanings are not fixed and are added to by outward influences. CSS has influenced the words and sentences of HTML from the outside. When an influence shifts the way a word is understood, this changes the message that is heard. The message heard is the DOM.

What is meant by boxifying design?

All forms of flat media tend to be box-shaped and can be broken down into boxes. Their wholeness is box shaped, examples of this are magazines, newspapers, photos, websites, tv and movies. Today most of these formats are displayed on screens, which in themselves are boxed. Boxifiying design is to take this initial box and dissect it further, forming more boxes. The way these corresponding boxes are arranged can be done to suit people's instinctive viewing tendencies, forming a hierarchy of content.

What is the box model?

The box model is a developers tool. The box model is a box that wraps all HTML elements. It is based on the values of content, padding, border and margin. Each of these has a corresponding area value, combining all these is the entire area of the box. Being able to control the way an element influences space is how the layout is formed.

  • Content - The content of the box, where text and images appear.
  • Padding - Clears an area around the content. The padding is transparent.
  • Border - A border that goes around the padding and content.
  • Margin - Clears an area outside the border. The margin is transparent.

Technical Blog Part 2 - Udacity Reflection

11th July, 2016


What is grid based design?

It is dividing up the space of a web page up with columns, columns run vertically. Designating a proportion of space to a column within CSS, allows you to use it as a class on HTML tags, which dictates how much screen width the HTML element will fill up. This makes it a lot easier to layout objects and changes how things will be displayed on small screens.

Whats all the hype about responsive web pages?

With more and more people using mobile devices to view and interact with websites. Websites that are able to adjust to the device that they are being viewed on, can ensure that users have positive interaction no matter what the device that is being used.

What is semantic structure?

A HTML semantic tag is one that denotes its content, such as img, a, p. Semantic structure takes this further and accepts that websites have shown over time to have universal attributes, such as, header, footer, aside, nav, section, article. This minimises the need to use divs with differing classes.
